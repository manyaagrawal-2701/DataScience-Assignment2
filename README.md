# Python Practice Questions (1–20)

This repository contains **20 Python questions** divided into four levels to strengthen core programming concepts.


## Easy (1–5)
Basic loops, conditionals, lists, tuples, and sets.

## Medium (6–10)
Nested loops, dictionaries, slicing, and set operations.

## Advanced (11–15)
Sorting with `lambda`, list comprehension, data processing, and duplicate removal.

## Tricky (16–20)
Random module, nested dictionaries, prime numbers, frequency counting, and tuple unpacking.
